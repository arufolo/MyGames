Re-Creation of the Axis flash game BomberBelts for the Game Artificial Intelligence course at UNC Charlotte

Assets folder contains the Source for the game including the Scripts my partner and I created
BomberBelts.exe is the executable version of the game

Controls:
Player 1 (Top Player): A-move Left, D-move right, S-send bomb
Player 2 (Bottom Player): Left Arrow-move left, Right Arrow-move right, Down Arrow-send bomb

This particular assignment was to recreate the flash game Bomber Belts, URL: http://axisgamesonline.com/games/13/

**The music was chosen by my partner as a joke, so my apologies.
